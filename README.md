### How to run?

Install dependencies

```
yarn || npm install
```

Development server

```
yarn dev || npm run dev
```

Production server

```
yarn server || npm run server
```

### How to deploy on vercel?

Use the vercel cli to deploy

```
yarn run vercel
yarn run vercel --prod
```
